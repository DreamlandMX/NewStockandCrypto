Test Checklist

  1. Register or log in first. Authentication is required before using notes and chat.
  2. Run database/supabase_schema.sql in Supabase to create the required tables.
  3. Click Test Connection to verify Supabase connectivity.
  4. Click Check Database Tables to confirm that all tables exist.
  5. Test note actions: create, fetch, and delete notes.
  6. Test chat actions: fetch boards, join a board, and send messages.
  7. Open two browser windows if you want to validate realtime subscriptions.
